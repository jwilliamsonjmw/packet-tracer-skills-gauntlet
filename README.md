# Packet Tracer Skills Gauntlet

A progressive 5-level Cisco Packet Tracer lab series built to develop 
and document hands-on networking skills toward CCNA certification.

## Levels

| Level | Topic | Status |
|-------|-------|--------|
| 1 | Basic LAN Setup | ✅ Complete |
| 2 | VLANs + Inter-VLAN Routing | ✅ Complete |
| 3 | OSPF Dynamic Routing | 🔒 Locked |
| 4 | ACLs + Network Security | 🔒 Locked |
| 5 | Full SOC-Ready Network | 🔒 Locked |

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI
- GitHub for documentation
